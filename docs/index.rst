He-Man's documentation!
=======================

Version: |version|

**He-Man** uses the *Empowering* Sword and does great things.

He-Man is a Empowering insight engine proxy which has results cached in a
MongoDB database.

Features
--------

:Authentification and security:
    To make API calls to He-Man you must have a token, and you can only view
    resources associated with this token. Eg. Reports only for your contracts.
:Insigth engine compatible API:
    Endpoints and resources are compatible with the Empowering API.
